# NoteCognition

NoteCognition is a minimalist, offline-first Markdown editor inspired by Notion, built with React, Dexie.js, and TailwindCSS.

## Features
- **Offline-First**: Powered by Dexie.js (IndexedDB) for persistent local storage.
- **Collaborative**: Ready for Dexie Cloud sync and real-time updates.
- **Minimalist Design**: Clean, distraction-free interface.
- **Live Preview**: Real-time markdown rendering.

## Tech Stack
- **Frontend**: React, Vite
- **Styling**: TailwindCSS, Framer Motion
- **Database**: Dexie.js, Dexie Cloud Addon
- **Icons**: Lucide React

## Running the code

Run `npm install` to install the dependencies.
Run `npm run dev` to start the development server.
